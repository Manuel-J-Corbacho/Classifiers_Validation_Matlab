# Classifiers_Validation_Matlab
Some Classifiers and several validation using matlab

Classifiers:
  K-means
  
  Euclidean Distance
  
  Nearest-neighbor
  
  K-nearest-neighbors
  
  Mahalanobis' Distance

Validations:
  Simple
  
  Resubstitution
  
  Cross-Validation
  
  Leave-One-Out
  
  Random-Sampling
  
  # Data Format
  
  The datas must be in a matrix, where each column is a item and every row an attribute of the items
